## Tabela Embarcações ->

![Embarcações](./images/Pagina-embarcacao.png "Embarcações")

- **Pesquisa:** Permite a Pesquisa através do Nome
- **Adionar Embarcação:** Permite adicionar uma nova embarcação
- **Exportar:** Permite exportar os dados para .CSV
- **Colunas:** ID, Nome, Tipo de Embarcação
- **Funções: ->**<br />
  &nbsp;&nbsp;&nbsp;&nbsp; - Editar Dados de uma Embarcação<br />
  &nbsp;&nbsp;&nbsp;&nbsp; - Eliminar a Embarcação<br />
  &nbsp;&nbsp;&nbsp;&nbsp; - Ao clicar, permite saber mais detalhes da embarcação<br />

### Detalhes da embarcação ->

![Embarcações](./images/detalhes-embarcacao.png "Embarcações")

- **Detalhes:**<br />
  &nbsp;&nbsp;&nbsp;&nbsp; - Descrição<br />
  &nbsp;&nbsp;&nbsp;&nbsp; - Observação<br />
  &nbsp;&nbsp;&nbsp;&nbsp; - Pessoas<br />
  &nbsp;&nbsp;&nbsp;&nbsp; - Viagens realizadas pela Embarcação<br />
- **Adicionar Proprietário:** Permite adicionar um novo Proprietário à embarcação escolhida<br />

## Tabela Tipos de Embarcações ->

![Embarcações](./images/Pagina-Tipoembarcacao.png "Embarcações")

- **Pesquisa:** Permite a Pesquisa através do tipo de embarcação
- **Adicionar Tipo:** Permite criar um novo Tipo de Embarcação
- **Exportar:** Permite exportar os dados para .CSV
- **Colunas:** Tipo
- **Funções: ->**<br />
  &nbsp;&nbsp;&nbsp;&nbsp; - Editar Dados de um Tipo de Embarcação<br />
  &nbsp;&nbsp;&nbsp;&nbsp; - Eliminar a Embarcação<br />
  &nbsp;&nbsp;&nbsp;&nbsp; - Ao clicar, permite saber mais detalhes da embarcação<br />

### Detalhes do Tipo de embarcação ->

![Embarcações](./images/detalhes-tipoembarcacao.png "Embarcações")

- **Detalhes:**<br />
  &nbsp;&nbsp;&nbsp;&nbsp; - Descrição<br />
  &nbsp;&nbsp;&nbsp;&nbsp; - Imagens<br />
- **Adicionar Imagem:** Permite adicionar uma imagem a partir de um URL<br />
